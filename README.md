# Local Hobbies

Mobile app that matches people by shared hobbies + location radius + availability.

## Tech Stack
- Mobile: React Native (Expo)
- API: Spring Boot
- DB: PostgreSQL

## Repo Structure
- apps/mobile — Expo app
- apps/api — Spring Boot API
- docs — schema + API contract
